UNDER THE HOOD

How RAG Works

The pipeline that makes AI answer from YOUR data — not hallucinate.

01
📄
Document Ingestion
Documents split into 200-character chunks using RecursiveCharacterTextSplitter.
02
🔢
Vectorization
Each chunk converted to numerical vectors using HuggingFace all-MiniLM-L6-v2 embeddings.
03
💾
Vector Storage
Vectors stored in ChromaDB — a specialized database for semantic similarity search.
04
🔍
Semantic Retrieval
User question vectorized, top-2 most similar chunks retrieved from ChromaDB.
05
🤖
Grounded Generation
Groq LLaMA 3.3 70B generates answer ONLY from retrieved chunks. Zero hallucination.
